Photone for OneDrive

Bienvenido a la página de soporte para Photone para OneDrive

Phontone le permite ver sus fotos de Microsoft onedrive en el último televisor de Apple

Consiste en una aplicación para iOS que se utiliza para iniciar sesión en onedrive y obtener un token de acceso (esto es válido para una hora) que luego se guarda en su cuenta de iCloud. Esto es necesario, ya que todavía no es posible iniciar sesión en onedrive directamente desde el Apple TV.

La principal aplicación es para Apple TV, que usando su cuenta de iCloud sincronizada es capaz de obtener el token de acceso que se puede utilizar para iniciar sesión en onedrive y descargar sus fotos.

El token de acceso sólo es válida durante una hora (la fecha de caducidad se muestra en la App IOS). Después de este tiempo tendrá que volver a autorizar a través de la aplicación de iOS.

Se puede quitar el token de acceso de cualquiera de las versiones de la aplicación. Si sales de iCloud el testigo permanece válido hasta la fecha de caducidad, pero si usted se inscribe como otro usuario de lo que será reemplazado.

NOTA

  • Se requiere una cuenta de Microsoft onedrive y cuenta de Apple iCloud.
  • Sólo onedrive personal, No Es compatible Negocios.
  • NOTA2: La aplicación iOS sólo se utiliza para autenticarse con onedrive, que no muestra ningún fotos.

Photone for OneDrive - Privacy Policy

  • No account details are stored by this app.
  • When authorizing from the iOS App an OAuth2 Token is obtained from Microsoft OneDrive.
    • This is stored in iCloud using the iCloud account that is currently signed in.
    • The token is made available to the corresponding Apple TV App.
    • As far as I am aware the token is stored securely using iCloud security.
    • The token supplied from Microsoft OneDrive is valid for an hour.
    • If iCloud on Apple TV is signed out of and/or the App is signed out of then the token is deleted.
  • As far as I am aware no Microsoft OneDrive nor iCloud account information is stored by either the iOS or Apple TV Apps. The sign-in process for Microsoft OneDrive does apparently leave a cookie on the device so when authorizing in the future the Microsoft Account login can be bypassed by Microsoft. This can be removed from the iOS App using the Sign Out button.
  • Photone (both the authorisation & viewer apps) does not track any usage of the App itself nor of any photos viewed.
